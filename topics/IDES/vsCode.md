|Action| Command|
|------|--------|
|Show command Palette | `Ctrl + Shift + P`
|Go to Definition | `F12`
|Go back | `Alt <-`
|Go forward | `Alt + ->`
|Show Terminal | `Ctrl + ' (quote)`
|New Terminal | `Ctrol + Shift + ' (quote)`
|Toggle Line Comment | `Ctrl + /`
|Toggle Block Comment | `Shift + Alt + A`
